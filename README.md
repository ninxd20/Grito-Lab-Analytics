# 📊 Multi-Channel Marketing Attribution & ROI Analytics

An interactive marketing analytics dashboard built with Streamlit to analyze campaign performance, compare attribution models, forecast revenue, and optimize marketing budgets through interactive visualizations.

## 🔗 Live Demo

https://ninad-grito-lab-analytics.streamlit.app/

## 🏆 Project Context

Developed for the **Grito Lab Challenge** under the **Techfest IIT Bombay Campus Ambassador Program 2026**.

## 🚀 Features

* Interactive Marketing KPI Dashboard
* Revenue, Spend & Conversion Analysis
* ROAS (Return on Ad Spend) Monitoring
* CPA (Cost Per Acquisition) Analysis
* CTR (Click-Through Rate) Tracking
* Last-Touch, Linear & Time-Decay Attribution Models
* Marketing Funnel Visualization
* Revenue Forecasting
* Budget Scenario Simulator
* Dataset Upload & Validation
* Budget Optimization Recommendations
* Downloadable Dataset Export

## ⭐ Highlights

* Built using Streamlit, Pandas, Plotly & Scikit-Learn.
* Implemented multiple attribution models for channel analysis.
* Added forecasting and budget simulation features.
* Supports custom dataset uploads with validation.
* Includes interactive charts, KPIs, and optimization insights.

## 📊 Key Metrics

* ROAS — Return on Ad Spend
* CPA — Cost Per Acquisition
* CTR — Click-Through Rate
* CPC — Cost Per Click
* ROI — Return on Investment
* Conversion Rate

## 🛠 Tech Stack

**Python • Streamlit • Pandas • NumPy • Plotly • Scikit-Learn**

## 📦 Installation

```bash
git clone https://github.com/NinadTarwate/grito-lab-analytics.git
cd grito-lab-analytics
pip install -r requirements.txt
streamlit run app.py
```
