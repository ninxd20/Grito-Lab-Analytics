# 📊 Multi-Channel Marketing Attribution & ROI Analytics

An enterprise-style marketing analytics dashboard that simulates multi-channel campaign performance across E-Commerce, Healthcare, and Mobility sectors. The platform evaluates attribution models, tracks key marketing KPIs, and provides data-driven budget optimization recommendations through interactive visualizations.

## 📦 Installation

### Clone Repository

```bash

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
